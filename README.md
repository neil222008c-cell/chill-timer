# Dimlight 🎬

**Movies for the mood you're in.** A cozy, late-night movie recommender that matches films to how much time you have and how awake you feel. Built as a Major Project.

> "I have exactly 90 minutes." · "Keep me awake." · "Something to fall asleep to." · "Just cozy."

## ✨ Features

- 🔐 **Supabase Authentication** — email/password + Google OAuth
- 🛡️ **Protected Routes** — `_authenticated` layout gates the app
- 🤖 **AI-powered recommendations** — Lovable AI (Gemini) picks a real film matching your mood, time, and energy
- 📚 **Full CRUD** on your saved history — mark watched, rate 1-5 stars, delete
- 📊 **Analytics Dashboard** — Recharts-powered stats on your watching patterns (mood, genre, activity)
- 🎨 **Cozy dim-lounge design** — warm amber tones, Fraunces + Inter, Framer Motion touches
- 🌐 **SEO ready** — proper meta tags, sitemap.xml, robots.txt

### Environment

The project reads Supabase config from `.env` (auto-generated by Lovable Cloud):

```
VITE_SUPABASE_URL=...
VITE_SUPABASE_PUBLISHABLE_KEY=...
SUPABASE_URL=...
SUPABASE_PUBLISHABLE_KEY=...
SUPABASE_SERVICE_ROLE_KEY=...   # server-only, injected at runtime
LOVABLE_API_KEY=...              # server-only, for AI Gateway
```

## 🗄️ Database schema

- `profiles` — auto-created on signup via `handle_new_user()` trigger
- `recommendations` — one row per saved pick (title, year, runtime, mood, energy, watched, rating)

Full RLS: every row scoped to `auth.uid()`.

## 🧠 How the recommender works

`src/lib/recommendations.functions.ts → pickMovie` posts to Lovable AI Gateway with a JSON-schema-constrained prompt. The model returns a single real movie matching:
- **mood** — `fast-paced` / `sleepy` / `exact-time` / `chill`
- **timeAvailable** — minutes
- **energy** — `wired` / `medium` / `sleepy`
- **exclude** — already-suggested titles this session (variety)

## 📝 Notes on requirements

- Uses **Lovable AI Gateway** in place of a raw Groq API key — same OpenAI-compatible interface, no key management needed. Swap the model id in `src/lib/ai-gateway.server.ts` to change providers.
- **Groq Vision** was scoped out (per project choice — vision doesn't fit a text-filter movie recommender).
- **Resend** integration is a follow-up: add a `RESEND_API_KEY` secret and wire an "email me this pick" server function to `src/lib/recommendations.functions.ts`.

## 🚢 Deploy

Click **Publish** in the Lovable editor (top-right) — the app deploys to `*.lovable.app` on Cloudflare Workers. Connect a custom domain in Project settings.
